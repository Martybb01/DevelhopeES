Alcuni utili tag inline sono il tag "cite", il tag "sup" o "sub" ... ah già, anche il tag "i" o "strong".

Come puoi vedere il tag

"p"

invece non è inline.
Ricorda di non mettere mai un tag p dentro l'altro. Usa invece il tag span o tag specifici.

Anche i tag main e aside sono dei tag block.
Se vuoi creare un layout dove aside sia effettivamente accanto al main dovrai infatti usare del CSS.
